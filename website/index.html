What you get

Language intelligence

Completions, hover, signature help, diagnostics, document & workspace symbols, and semantic highlighting — via the bundled plc-lsp-server.

Run Structured Text

Execute a .st file from the command, the editor title-bar Run button, or a right-click — through the bundled scan-cycle runtime.

Full stepping debugger

Breakpoints, step over/into/out, live Variables (scalars + function-block outputs) and Call Stack. F5 just works; right-click Debug too.

Self-contained

Each platform package bundles the native compiler + language server. No Rust toolchain, no separate downloads — install and go.

Install

  1. Download the package for your platform above.
  2. From a terminal: 
    code --install-extension plc-vscode-<platform>.vsix
    …or in VS Code: Extensions view → ⋯ → Install from VSIX…
  3. Open any .st / .iecst / .plcst file. Press F5 to debug, or right-click → Run.
